# Ashley Fernander — Personal Portfolio

Personal portfolio and resume site for Ashley Fernander, live at [ashleyfernander.com](https://ashleyfernander.com).

## About

Built with HTML, CSS, and Bootstrap 5. Hosted on GitHub Pages with a custom domain via GoDaddy.

## Structure

```
My-Website/
├── index.html       # Main site
├── img/             # Images and assets
└── CNAME            # Custom domain config
```

## Local Development

No build step needed. Open `index.html` directly in a browser, or use the VS Code Live Server extension for hot reload.

## Deployment

Pushes to `main` deploy automatically via GitHub Pages.
